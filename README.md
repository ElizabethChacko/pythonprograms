## Prerequisites

- Python 3.x

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/ElizabethChacko/pythonprograms.git
    cd pythonprograms
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run a program:
    ```bash
    python src/basicprograms/your_script.py
    ```
